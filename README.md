# 📊 Customer Shopping Behavior Analysis

## 📌 Overview

This project focuses on analyzing customer shopping behavior using data analytics techniques. It includes data cleaning, exploratory data analysis (EDA), SQL-based querying, and visualization through a Power BI dashboard. The goal is to extract meaningful insights that can help businesses understand customer patterns and improve decision-making.

---

## 📂 Dataset

* **Name:** Customer Shopping Behavior Dataset
* **Format:** CSV
* **Contents:**

  * Customer demographics
  * Purchase history
  * Product categories
  * Transaction details

---

## 🛠 Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* Jupyter Notebook
* PostgreSQL / MySQL / SQL Server
* Power BI
* Git & GitHub

---

## 🚀 Steps Involved

### 1. Data Loading

* Imported dataset using Pandas
* Verified structure and column details

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Corrected data types

### 3. Exploratory Data Analysis (EDA)

* Analyzed customer trends
* Identified top-selling categories
* Visualized spending patterns

### 4. SQL Analysis

* Imported data into SQL database
* Performed queries like:

  * Total sales
  * Customer segmentation
  * Category-wise revenue

### 5. Dashboard Creation

* Built an interactive dashboard in Power BI
* Visualized KPIs like:

  * Total Revenue
  * Customer Count
  * Sales Trends

---

## 📊 Dashboard

* Interactive Power BI dashboard
* Filters for category, customer type, and time
* Visual insights for quick decision-making

## 📈 Results & Insights

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone <your-repo-link>
   ```

2. Open Jupyter Notebook:

   ```bash
   jupyter notebook
   ```

3. Run the notebook:

   * Open `.ipynb` file
   * Execute cells step-by-step

4. SQL Setup:

   * Import dataset into your SQL database
   * Run provided queries

5. Power BI:

   * Open `.pbix` file
   * Refresh data if needed
